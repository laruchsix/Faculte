Module Btree

module Btree: sig .. end
Minimal binary tree for AP3 course
Author(s): Laurent Fuchs

This library follows the usual abstract data type except for the utilities functions
type 'a t_btree
type for polymorphic binary trees

Constructor functions

val empty : unit -> 'a t_btree
Empty tree constant
val rooting : 'a * 'a t_btree * 'a t_btree -> 'a t_btree
Rooting of a new tree

Selector functions

val root : 'a t_btree -> 'a
Value of the root label

Semantics

  • root(empty()) = Invalid_argument
  • root(rooting(v, l, r)) = v

Raises Invalid_argument when the tree is empty
val lson : 'a t_btree -> 'a t_btree
Left son

Semantics

  • lson(empty()) = Invalid_argument
  • lson(rooting(v, l, r)) = l

Raises Invalid_argument when the tree is empty
val rson : 'a t_btree -> 'a t_btree
Right son

Semantics

  • rson(empty()) = Invalid_argument
  • rson(rooting(v, l, r)) = r

Raises Invalid_argument when the tree is empty

Predicate

val isEmpty : 'a t_btree -> bool
Semantics
  • isEmpty(empty()) = true
  • isEmpty(rooting(v, l, r)) = false
