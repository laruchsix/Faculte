Module Gtree

module Gtree: sig .. end
Minimal general tree for AP3 course
Author(s): Laurent Fuchs

This library follows the usual abstract data type except for the utilities functions
type +'a t_gtree
type for polymorphic general trees
type +'a t_forest
type for polymorphic forest of general trees

Constructor functions

val f_empty : unit -> 'a t_forest
Empty forest constant
val rooting : 'a * 'a t_forest -> 'a t_gtree
Rooting of a new tree
val insert : 'a t_gtree * 'a t_forest -> 'a t_forest
Inserting a tree into a forest to get a new forest

Selector functions

val root : 'a t_gtree -> 'a
Value of the root label

Semantics

  • root(rooting(v, f)) = v

val sons : 'a t_gtree -> 'a t_forest
Sons of the root of a general tree

Semantics

  • sons(rooting(v, f)) = f

val eldest : 'a t_forest -> 'a t_gtree
Eldest son of a forest

Semantics

  • eldest(empty()) = Invalid_argument
  • eldest(insert(t, f)) = t

Raises Invalid_argument when the forest is empty
val brothers : 'a t_forest -> 'a t_forest
Brothers of eldest son of a forest. This is the forest without the eldest son.

Semantics

  • brothers(empty()) = Invalid_argument
  • brothers(insert(t, f)) = f

Raises Invalid_argument when the forest is empty

Predicate

val f_isEmpty : 'a t_forest -> bool
Semantics
  • f_isEmpty(empty()) = true
  • f_isEmpty(insert(t, f)) = false
